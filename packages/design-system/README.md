# @fabrica/design-system

**CA-001** · Sistema de diseño compartido de la Fábrica de Software Académico.

Provee los *design tokens* (`tokens.json`) — paleta académica, tipografía Inter,
radios y sombras — y las clases base (formularios, botones, tablas, badges,
glassmorphism) que todos los productos derivados reutilizan para una identidad
visual consistente.

```js
const tokens = require('@fabrica/design-system/tokens');
// tokens.color.primary === '#2563eb'
```
