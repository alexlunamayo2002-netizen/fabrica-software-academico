# @fabrica/angular-auth

Core Assets de **autenticación y autorización** para el frontend Angular.

| ID | Asset |
|----|-------|
| CA-002 | Modelo de Usuario (frontend) |
| CA-003 | AuthService (login/registro/JWT) |
| CA-004 | AuthGuard (rutas protegidas) |
| CA-005 | RoleGuard (autorización por rol) |
| CA-006 | Componente de Login |
| CA-007 | Registro abierto (variabilidad) |

Los productos derivados importan estos componentes/servicios en lugar de
copiarlos, garantizando una única fuente de verdad para la seguridad.
